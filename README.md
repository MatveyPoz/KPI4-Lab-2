# KPI4-Lab-2
 Lab work N2 - Simple Calculatore

## IM-32 Pozdieiev Matvii

### How to install:
1. Clone the repository: `git clone https://github.com/your_username/KPI4-Lab-1.git`
2. Navigate to the project directory: `cd KPI4-Lab-2`
3. Create a virtual environment: `python -m venv venv`
4. Activate the virtual environment: `source venv/bin/activate` on Unix or `venv\Scripts\activate` on Windows
5. Install requirements: `pip install -r requirements.txt`
6. Run `python -m game_of_life`
7. Run tests: `python -m pytest --cov=game_of_life -q` or `python -m pytest --cov=game_of_life --cov-report=term-missing`
